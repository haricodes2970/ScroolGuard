# ScrollGuard

Android app that blocks vertical scrolling in **Instagram Reels** and **YouTube Shorts** only. Stories, DMs, search, and taps work normally everywhere else.

## Requirements

- Android 7.0+ (API 24)
- Physical device recommended for testing

## Setup

1. Build and install from Android Studio or `./gradlew installDebug`
2. Enable **ScrollGuard** under Settings → Accessibility
3. Open the app and turn on **Enable Scroll Blocking**
4. Optionally set a grace-period timer (free scrolling minutes before lock)

## Privacy

No internet permission. No analytics. All processing is on-device.
